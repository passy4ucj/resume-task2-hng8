# DevCamper API

> HNG-8 Task2 Resume Form

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run server

# Run in prod mode
npm start
```
